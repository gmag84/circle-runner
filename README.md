A fast-paced browser game where you dodge incoming circles and survive as long as possible.

## How to Play
- **WASD** or **Arrow Keys** to move your circle
- Dodge all incoming colored circles
- The longer you survive, the faster they get!
- Press **Space** to start / restart

## Run Locally
```bash
npm install
npm run dev
```
